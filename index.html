Conexión Google Sheets
Instrucciones para conectar Google Sheets
  1. Crea un nuevo Google Sheet (puede tener cualquier nombre)
  2. Ve a Extensiones → Apps Script
  3. Borra el código existente y pega el código que aparece abajo
  4. Haz clic en Implementar → Nueva implementación
  5. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquiera
  6. Autoriza los permisos y copia la URL del Web App
  7. Pega la URL arriba y guarda
