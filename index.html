Conexión Google Sheets
Instrucciones para conectar Google Sheets
  1. Crea un nuevo Google Sheet (puede tener cualquier nombre)
  2. El Apps Script crea automáticamente las pestañas necesarias al guardar por primera vez
  3. Pestañas que se crean solas: Datos generales personal · Detalle de asistencia · Vacaciones · Resumen Mensual · Tardanzas Descuentos
  4. ✓ Todo lo que guardes en la plataforma se sincroniza automáticamente si hay conexión activa
  5. Ve a Extensiones → Apps Script
  6. Borra el código existente y pega el código que aparece abajo
  7. Haz clic en Implementar → Nueva implementación
  8. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquiera
  9. Autoriza los permisos y copia la URL del Web App
  10. Pega la URL arriba y guarda
